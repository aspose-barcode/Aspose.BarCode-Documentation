{{% alert color="primary" %}} 

Aspose.BarCode for JasperReports supports more than 25 barcode symbologies used in a variety of industries. These include linear as well as 2D barcode symbols.

{{% /alert %}} 
### **Supported Barcode Symbologies**
- Australia Post
- Aztec
- BooklandEAN (ISBN)
- Codabar
- Code39
- Code93
- Code11
- Code128
- Datamatrix
- EAN 8
- EAN 13
- EAN 14 (SCC14)
- EAN 128
- Identcode
- Interleaved 2 of 5
- Itf14
- Leticode
- MacroPdf417
- Matrix 2 of 5
- MSI
- OneCode
- OPC
- Pdf417
- Planet
- Postnet
- QR
- SSCC 18
- Standard 2 of 5
- Supplement Code +2 (For EAN and UPC code)
- Supplement Code +5 (For EAN and UPC code)
- SwissQR
- SVG
- UPC A
- UPC E
- VIN

# Visual Parameters & Layout

## Overview

After you select a symbology and set the code text, you typically need to control layout and rendering settings such as size, text, colors, and rotation.

In Aspose.BarCode for Java, most visual settings are configured through `generator.getParameters()`.

The pages below focus on the parameters that affect the final geometry and the resulting image.

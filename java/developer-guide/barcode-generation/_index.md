# Barcode Generation

This section explains how to generate barcodes with Aspose.BarCode for Java.

## What you can do here

You can generate a barcode by selecting a symbology (`EncodeTypes`), providing the value to encode, tuning visual parameters (size, colors, text, rotation), and saving the result to a file or stream.

## Typical workflow

1. Create a `BarcodeGenerator` with `EncodeTypes` and code text.
2. Configure geometry and layout (X-Dimension, bar height, padding, AutoSizeMode).
3. Configure visual settings (code text, captions, colors, rotation).
4. Save the result to a file or an `OutputStream`.

# Supported Complex Barcode Types

Aspose.BarCode for Java provides typed APIs for several complex barcode standards used in payment, healthcare, postal, shipping, and identification workflows.

Each article in this section explains how to create the corresponding structured data model, generate its carrier barcode, recognize the image, and decode the payload back into a typed Java object.

This section contains the following articles:

- [Swiss QR Barcodes](swiss_qr/) — generate Swiss QR Bills and decode structured payment data.
- [HIBC Barcodes](hibc/) — work with HIBC LIC product data and HIBC PAS healthcare records.
- [Royal Mail Mailmark Barcodes](royal_mail_mailmark/) — generate and read Mailmark 1D and Mailmark 2D postal barcodes.
- [MaxiCode Barcodes](maxicode/) — create and decode structured MaxiCode mode 2 and mode 3 payloads.
- [USA Driver ID Barcodes](usa_driver_id/) — generate and read AAMVA driver-license data stored in PDF417.
